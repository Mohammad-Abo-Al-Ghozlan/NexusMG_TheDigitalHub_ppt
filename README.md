# NexusMG The Digital Hub Presentation

This project is a Next.js presentation app for NexusMG, an AI-powered platform that evaluates developer readiness from LinkedIn, GitHub, and CV data.

## Overview

- Built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.
- The presentation is implemented in `components/presentation/slides.tsx`.
- Slide text content is also available in `slides-content.md`.

## Run Locally

```bash
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev` - start the development server
- `pnpm build` - build the app for production
- `pnpm start` - run the production build
- `pnpm lint` - run ESLint
